Java Web基础知识点

2017-09-25
Java
JSP

Java Web 的一些小知识点。

1、jsp的组成元素?

  1. 静态内容
  2. 指令(<%@ %>)
  3. 表达式(<%=表达式%>，输出)
  4. 小脚本(<%Java代码%>)
  5. 声明(<%! %>)
  6. 动作(<jsp:动作名></jsp:动作名>)
  7. 注释(<!-- -->客户端可见，<%– –%>客户端不可见)

2、jsp的隐式对象?

  1. 输入输出对象：request、response、out
  2. 作用域通信对象：pageContext、request、session、application
  3. Java对象：page、config
  4. 错误对象：exception

3、转发和重定向的区别以及各自的语句?

区别：

  1. 转发在服务器端完成；重定向在客户端完成；
  2. 转发速度快；重定向速度慢；
  3. 转发是同一次请求；重定向是两次不同请求；
  4. 转发的地址栏没变化；重定向的地址栏有变化；
  5. 转发必须在同一服务器下完成；重定向可以在不同服务器下完成。

语句：

  1. 转发：request.getRequestDispatcher("页面").forward(request, response);
  2. 重定向：response.sendRedirection("页面");

4、get和post的区别？

  1. get请求：相当于明信片，所有的信息都显示在地址栏；请求的信息有限，而且只能传递字符串；方便。
  2. post请求：相当于信封，安全性高；大小没有限制。

5、默认端口？

  1. tomcat : 8080
  2. http : 80
  3. mysql ：3306
  4. sqlserver : 1433
  5. oracle : 1521

6、会话跟踪的原理？

       对一个用户对服务器连续的请求和接收响应进行监视。

7、session和cookie的区别？

  1. cookie数据存放在客户的浏览器上；session数据放在服务器上；
  2. cookie不是很安全，别人可以分析存放在本地的cookie并进行cookie欺骗，考虑到安全应当使用session；
  3. session会在一定时间内保存在服务器，当访问增多，会比较占用服务器性能，考虑到减轻服务器性能方面，应当用cookie；
  4. 单个cookie最大4k，很多浏览器都限制一个站点最多存20个cookie；
  5. 建议：登录信息等重要信息存session，其他需保留信息放在cookie。

8、jsp的三大指令？

  1. page
  2. taglib
  3. include

9、实现会话跟踪的四种方式？

  1. URL重写
  2. 隐藏表单域
  3. session
  4. cookie

10、b/s和c/s的区别？

  1. c/s是客户端/服务器，一般要在电脑安装软件，或运行一个客户端软件才能使用，在系统升级时稍麻烦点。
  2. b/s是浏览器/服务器，打开浏览器就能使用，一般是网页形式，与c/s相比，web应用程序易维护、升级和部署，速度相对较慢，因为数据全保存在服务器上。

11、getParameter()和getAttribute()的区别？

  1. getPatameter()是用来接收用post和get方式传递过来的参数的；getAttribute()必须先setAttribute();
  2. getParameter()传递的数据会从web客户端传到web服务器端，代表HTTP请求数据，返回String类型的数据。setAttribute()和getAttribute()传递的数据只会存在于web容器内部。
