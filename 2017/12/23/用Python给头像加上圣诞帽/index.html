用Python给头像加上圣诞帽

2017-12-23
Python
Python

引言

随着圣诞的到来，大家纷纷@官微给自己的头像加上一顶圣(yuan)诞(liang)帽。当然这种事情用很多P图软件都可以做到。但是作为一个学习图像处理的技术人，还是觉得我们有必要写一个程序来做这件事情。而且这完全可以作为一个练手的小项目，工作量不大，而且很有意思。

用到的工具

OpenCV库：安装方法–pip install opencv-python -i http://pypi.douban.com/simple/

dlib库：安装方法–pip install dlib -i http://pypi.douban.com/simple/

图片素材在最下面的Github项目里都有。
ps:在国内用豆瓣源速度刚刚的。
